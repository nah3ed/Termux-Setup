# Termux-Setup-Script
This Repo is for testing purposes only

## How to install
Install git
```pkg install git```

Clone the repository
```git clone https://github.com/nah3ed/Termux-Setup.git

Change directory to the cloned repository
```cd Termux-Setup

Change access permission
```chmod +x setup.sh```


Execute the program
```./setup.sh```
